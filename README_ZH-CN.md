# Genshin.Downloader
从对应的官方服务器下载原神的游戏资源文件。

## 注意事项
1. 这个程序只有中文版本。
1. ***!!仅供 64 位 Windows 系统使用!!***
1. 本程序由我独自开发和维护，属于自用工具分享，故**兼容性&可用性无法保证**。
